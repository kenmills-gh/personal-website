# Portfolio & Labs Repository

## 1. Project Overview

This repository serves as a centralized hub containing completed engineering lab documents, standalone source code scripts, and a responsive web portfolio showcasing major projects.

### Current Project Inventory

- **Personal Portfolio (`index.html` / `style.css`)**: A modern, responsive developer landing page featuring a component-driven CSS grid and optimized media queries for mobile layouts.
- **JavaScript Calculator (`index.js`)** : A structural arithmetic engine featuring standalone functional logic, automated operation state history tracking, and zero-division exception handling.
- **OpsTrack API Gateway(`server.js`)**: A backend architectural platform utilizing structured relational record tracking and secure session data processing.

---

## 2. System Requirements & Troubleshooting

- **Runtime Environment:** Node.js (v18+ recommended) or any modern web browser viewport.

- **OS Compatibility:** Native execution via Ubuntu Linux CLI / VS Code Integrated Terminal.

- **Broken Image Fix:** If project screenshots fail to display on local live views, verify that file names match the explicit paths inside index.html and that image files do not contain spaces.
